# Real Estate Price Prediction Project
Developed an end-to-end Machine Learning solution for predicting Bengaluru house prices using advanced regression models. Implemented data preprocessing, feature engineering, exploratory data analysis, model optimization, and performance evaluation to deliver accurate and reliable price predictions.

## Objective

The objective of this project is to predict real estate prices using Machine Learning algorithms.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- XGBoost

## Machine Learning Algorithms

- Linear Regression
- Random Forest Regressor
- XGBoost Regressor

## Features Used

- Area Type
- Location
- Total Square Feet
- BHK
- Bathrooms
- Balcony

## Steps Performed

1. Data Collection
2. Data Cleaning
3. Handling Missing Values
4. Feature Engineering
5. Data Visualization
6. Model Training
7. Model Evaluation
8. Model Saving

## Evaluation Metrics

- MAE
- MSE
- RMSE
- R2 Score

## Visualizations

- Correlation Heatmap
- Price Distribution

## Conclusion

XGBoost achieved the best performance among all models due to better handling of complex relationships in the dataset.
